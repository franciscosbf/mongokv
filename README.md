# MongoDB K/V Store

PostgreSQL extension to manage MongoDB collections in a key-value store fashion (only some types).

## Development Tools

- [GNU make](https://www.gnu.org/s/make/manual/make.html)
- [pkg-config](https://www.freedesktop.org/wiki/Software/pkg-config/)

## Dependencies

- [PostgreSQL 16.3](https://www.postgresql.org/)
- [MongoDB C Driver 1.28.1](https://mongoc.org/)
- [GLib 2.0](https://docs.gtk.org/glib/index.html)
